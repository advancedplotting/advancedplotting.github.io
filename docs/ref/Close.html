Close Plot ID¶

Close a plot identifier created with New Plot ID. Any further use of the identifier will result in an error.

To simplify your cleanup code, this VI ignores invalid Plot ID inputs. Like other LabVIEW “close” VIs, this VI will run normally, and the plot will be closed, even if errors are present on the “error in” terminal.

Plot ID
Plot identifier to close. An invalid or already-closed identifier will be ignored.
Error In/Out
Errors will be propagated from error in to error out, but are otherwise ignored by this VI.

Errors¶

None.
